# Exercícios de Lógica de Programação em C#

Repositório desenvolvido para praticar lógica de programação e consolidar os principais fundamentos da linguagem C#.

---

## Sobre o projeto

Este repositório foi criado como parte da minha jornada de estudos em desenvolvimento de software, com o objetivo de consolidar os principais conceitos da programação antes da construção de aplicações mais complexas.

Os exercícios estão organizados por níveis de dificuldade, permitindo uma evolução gradual desde os conceitos básicos até Programação Orientada a Objetos.

Além das soluções implementadas, este repositório representa minha evolução prática durante os estudos e faz parte da construção do meu portfólio como desenvolvedora.

---

## Conteúdo

O projeto está organizado em níveis de aprendizagem, contemplando os seguintes conteúdos:

- Nível 1 — Entrada, processamento e saída de dados
- Nível 2 — Estruturas condicionais
- Nível 3 — Estruturas de seleção
- Nível 4 — Estruturas de repetição
- Nível 5 — Vetores e matrizes
- Nível 6 — Métodos
- Nível 7 — Programação Orientada a Objetos
- Nível 8 — Coleções
- Nível 9 — Manipulação de arquivos
- Nível 10 — Tratamento de exceções

## Progresso

A evolução deste repositório pode ser acompanhada através dos arquivos `README.md` presentes em cada nível, onde são documentados os exercícios, os conteúdos abordados e o andamento dos estudos.

## Estrutura do Repositório

Cada nível possui uma pasta própria contendo:

- README.md com a descrição do módulo;
- Exercícios propostos;
- Soluções desenvolvidas em C#;
- Documentação complementar, quando aplicável.

---

## Como Executar

1. Clone o repositório para a sua máquina local.
2. Abra o arquivo de solução (`.sln`) no Visual Studio.
3. Pressione **F5** ou clique no botão **Play** no topo do Visual Studio para executar o projeto.
4. No console (na tela preta que vai abrir), **digite o número do exercício que você gostaria de testar** e pressione **Enter**.
