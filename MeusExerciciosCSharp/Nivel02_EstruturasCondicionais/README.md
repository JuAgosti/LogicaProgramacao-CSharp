# 📘 Nível 2 — Estruturas Condicionais

Este nível aprofunda os conceitos de lógica de programação em C#, introduzindo as estruturas condicionais utilizadas para tomada de decisões durante a execução de um programa.

Os exercícios abordam comparações, validações, classificações, operações condicionais e diferentes fluxos de execução utilizando `if`, `else` e `switch`.

---

## 🎯 Objetivos

Ao concluir este nível, espera-se que o estudante seja capaz de:

- Utilizar estruturas condicionais (`if`, `else` e `switch`).
- Comparar valores utilizando operadores relacionais.
- Aplicar operadores lógicos.
- Validar dados informados pelo usuário.
- Resolver problemas utilizando tomada de decisão.
- Classificar informações de acordo com determinadas regras.
- Desenvolver algoritmos com diferentes caminhos de execução.

---

## 📚 Conteúdos abordados

- Estruturas condicionais
- Operadores relacionais
- Operadores lógicos
- Comando `if`
- Comando `else`
- Comando `switch`
- Validação de dados
- Comparação de valores
- Classificação de informações
- Resolução de problemas

# 📝 Exercícios

## Exercício 11 — Troca de Valores

Leia dois valores inteiros para as variáveis **A** e **B**.

Efetue a troca dos valores entre as variáveis, de forma que:

- A passe a possuir o valor de B.
- B passe a possuir o valor de A.

Ao final, exiba os valores após a troca.

---

## Exercício 12 — Módulo de um Número

Leia um número inteiro e exiba seu módulo (valor absoluto).

Considere:

```text
Se x ≥ 0 → módulo = x

Se x < 0 → módulo = x × (-1)
```

---

## Exercício 13 — Ordem Decrescente

Leia três números inteiros e apresente-os em ordem decrescente.

---

## Exercício 14 — Diferença entre Dois Números

Leia dois números inteiros e apresente a diferença entre o maior e o menor valor.

---

## Exercício 15 — Média Escolar

Leia quatro notas escolares de um aluno.

Calcule a média.

- Se a média for maior ou igual a **7**, informar que o aluno foi aprovado.
- Caso contrário, solicitar a nota da recuperação.
- Calcular uma nova média utilizando a média anterior e a nota da recuperação.
- Se a nova média for maior ou igual a **7**, informar que o aluno foi aprovado na recuperação.
- Caso contrário, informar que o aluno foi reprovado.

Em todos os casos, apresentar o valor da média obtida.

---

## Exercício 16 — Maior e Menor Número

Leia dois números inteiros.

O programa deverá informar:

- O maior número.
- O menor número.

Caso os dois números sejam iguais, informar essa condição.

---

## Exercício 17 — Validação de Faixa

Leia um número inteiro.

Verifique se ele pertence ao intervalo:

```text
0 até 9
```

Exiba:

- **Valor válido**, caso esteja dentro da faixa.
- **Valor inválido**, caso contrário.

---

## Exercício 18 — Código por Extenso

Leia um número inteiro representando um código.

Utilizando o comando **switch**, exiba:

| Código | Resultado |
|--------:|-----------|
| 1 | Um |
| 2 | Dois |
| 3 | Três |

Caso o código seja diferente desses valores, informar:

```text
Código inválido
```

---

## Exercício 19 — Classificação de Triângulo

Leia os três lados de um triângulo.

Primeiramente, verifique se os valores formam um triângulo.

Caso formem, classifique-o como:

- Equilátero
- Isósceles
- Escaleno

Caso contrário, informar que os valores não representam um triângulo válido.

---

## Exercício 20 — Menor e Maior Valor

Leia três números inteiros maiores que zero.

Caso algum valor seja inválido, solicitar a digitação novamente.

Ao final:

- Exiba o menor valor multiplicado pelo maior.
- Exiba o maior valor dividido pelo menor.

---

## Exercício 21 — Número Positivo ou Negativo

Desenvolva um programa que leia um número inteiro e informe se ele é:

- Positivo
- Negativo

Após cada execução, permita que o usuário decida se deseja continuar utilizando o programa.

---

## Exercício 22 — Armazenando Valores

Leia um número.

- Se o número for positivo, armazená-lo na variável **A**.
- Se o número for negativo, armazená-lo na variável **B**.

Ao final, exiba o valor armazenado.
